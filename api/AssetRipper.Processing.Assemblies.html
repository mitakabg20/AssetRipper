Namespace AssetRipper.Processing.Assemblies

Classes

AttributePolyfillGenerator

Generates polyfills for missing attributes using the Roslyn compiler. During compilation, these attributes can be stripped because they're not "custom" attributes. They're special and get encoded in the metadata directly.

ForwardingAssemblyGenerator

ILSpy needs System.Runtime.CompilerServices.Unsafe and System.Runtime.InteropServices to decompile some code. This generates these assemblies with forwarders to the mscorlib types.

MethodStubbingProcessor

This processor replaces the method bodies of all methods with a minimal implementation.

MonoExplicitPropertyRepairProcessor

Fixes explicit property implementations in Mono assemblies.

NullableRemovalProcessor

This processor removes nullable annotations from all locations.

NullRefReturnProcessor

Returning a null by reference causes an error before C# 11. This replaces the method body with a throw statement.

ObfuscationRepairProcessor

Improves decompilation of obfuscated assemblies.

RemoveAssemblyKeyFileAttributeProcessor

Removes System.Reflection.AssemblyKeyFileAttribute from assemblies. It causes compile errors because the key file is not present.

SafeAssemblyPublicizingProcessor

This processor makes (nearly) all types, methods, and fields public in the assembly. It handles edge cases like determing if something should be publicized and adding [NonSerialized] to newly public instance fields.

UnmanagedConstraintRecoveryProcessor

Recover unmanaged constraints for generic parameters in Il2Cpp games. This prevents compile errors when using pointers to generic parameters.
